# KTrek-backed-database

Общая база данных для сервисов KTrek

## Запуск

1. `npm i`

2. `cp .env.template .env`

3. `npm run start`

### Дополнительный функционал

Запуск

```bash
npm run start
```

Остановка без удаления

```bash
npm run stop
```

Удаление(можно запускать при работающей бд)

```bash
npm run remove
```
